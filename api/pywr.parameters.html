pywr.parameters.licenses module¶

class pywr.parameters.licenses.AnnualExponentialLicense(amount, max_value, k=1.0)¶

Bases: pywr.parameters.licenses.AnnualLicense

An annual license that returns a value based on an exponential function of the license’s current state.

The exponential function takes the form,

\[f(t) = \mathit{max_value}e^{-x/k}\]

Where \(x\) is the ratio of actual daily averaged remaining license (as calculated by AnnualLicense) to the expected daily averaged remaining licence. I.e. if the license is on track the ratio is 1.0.

class pywr.parameters.licenses.AnnualHyperbolaLicense(amount, value)¶

Bases: pywr.parameters.licenses.AnnualLicense

An annual license that returns a value based on an hyperbola (1/x) function of the license’s current state.

The hyperbola function takes the form,

\[f(t) = \mathit{value}/x\]

Where \(x\) is the ratio of actual daily averaged remaining license (as calculated by AnnualLicense) to the expected daily averaged remaining licence. I.e. if the license is on track the ratio is 1.0.
