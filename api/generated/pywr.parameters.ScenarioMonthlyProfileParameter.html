pywr.parameters.ScenarioMonthlyProfileParameter¶

class pywr.parameters.ScenarioMonthlyProfileParameter¶

Parameter which provides a monthly profile per scenario

Behaviour is the same as MonthlyProfileParameter except a different profile is returned for each ensemble in a given scenario.

See also

MonthlyProfileParameter, ArrayIndexedScenarioMonthlyFactorsParameter
